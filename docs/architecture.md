# 拼省省系统架构

## 服务职责

| 服务 | 默认端口 | 当前职责 |
|---|---:|---|
| Gateway | 8080 | 统一入口、路由和跨域 |
| Auth | 9000 | 登录、令牌和用户身份 |
| System | 9001 | 用户、角色和权限 |
| Product | 8081 | 商品、分类、SKU 和商品详情 |
| Cart | 8082 | 购物车和购物车选中状态 |
| Order | 8083 | 地址、结算、订单和订单明细 |
| Payment | 8084 | 模拟支付和支付状态 |
| Frontend | 5173 | Vue 3 用户端 |

## 业务主链路

```text
用户登录 -> 浏览商品 -> 商品详情 -> 加入购物车
    -> 结算 -> 锁定库存 -> 创建订单 -> 模拟支付
```

第 1 天先验证商品服务和前端骨架；后续服务按这条主链路逐步加入，不提前虚构未实现功能。
